# Specs

**One folder per feature:** `YYYY-MM-DD-<slug>/` containing `spec.md`, `plan.md`, `tasks.md`.

To start: copy the three files from `../templates/` into a new dated folder, then follow `../README.md`.

```
specs/
├── 2026-06-10-add-trade-journal/
│   ├── spec.md
│   ├── plan.md
│   └── tasks.md
└── 2026-06-12-meeting-scribe-mode/
    ├── spec.md
    ├── plan.md
    └── tasks.md
```

Real worked example: the Knowledge Curator build at `../../schema/specs/`.

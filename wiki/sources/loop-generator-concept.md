# Loop Generator App Concept

## Source

- **File:** `2026-06-09 Loop generator app asks AI to define yesno success criteria.md`
- **Date:** 2026-06-09 | **Duration:** 1m29s

## Core Content

Early voice note concept for what became the agent-ready-coding-loop. The agent asks the user about their project and goals, then helps define binary yes/no success criteria. Examples given: PDF→Markdown conversion, drag-and-drop file processing. Success criteria must be unambiguous (yes/no, no in-between).

## Main Points

- Core pattern: AI interviews user → generates binary success criteria → drives a build loop
- Criteria must be yes/no: "the software will produce X" not "the software will do well"
- This concept was later formalised as the Loop Generator / agent-ready-coding-loop v2.2

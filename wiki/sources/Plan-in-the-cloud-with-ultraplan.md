## Source
- Original file: [[raw/Plan in the cloud with ultraplan.md]]

## Core Content
This source introduces [[entities/ultraplan|ultraplan]], a feature integrated within [[entities/Claude-Code|Claude Code]] that facilitates a [[concepts/Hybrid-Planning-Workflow|Hybrid Planning Workflow]]. The tool allows developers to initiate project plans via a Command Line Interface (CLI), transition to a web-based interface for more detailed drafting and refinement, and finally execute the resulting plan either in a remote cloud environment or locally within their terminal. This seamless integration aims to combine the efficiency of terminal operations with the visual capabilities of a web GUI.

## Main Points
- **Cross-Platform Transition**: Enables a fluid move between CLI-based planning and web-based drafting.
- **Flexible Refinement**: Plans started in the terminal can be visually refined on the [[entities/Claude-Code|Claude Code]] web interface.
- **Versatile Execution**: Supports executing finalized plans both remotely in the cloud and locally in the terminal.

# Claude Code

## Basic Information
- Type: product
- Source: [[sources/Plan-in-the-cloud-with-ultraplan|Plan-in-the-cloud-with-ultraplan]]

## Description
Claude Code is a developer tool providing a hybrid workflow through both a Command Line Interface (CLI) and a web-based interface for coding and planning. It serves as the primary host environment for the [[entities/ultraplan|ultraplan]] feature, enabling users to manage the entire lifecycle of a development plan. The product allows for seamless transitions where work can be initiated in a local terminal and refined in a cloud-hosted web application, ensuring consistent execution across different environments.

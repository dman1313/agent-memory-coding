# ultraplan

## Basic Information
- Type: product
- Source: [[sources/Plan-in-the-cloud-with-ultraplan|Plan-in-the-cloud-with-ultraplan]]

## Description
ultraplan is a specialized tool integrated with [[entities/claude-code|Claude Code]] designed for cloud-based project planning. It enables developers to bridge the gap between local terminal operations and web-based drafting by allowing a plan to be initiated in the CLI and transitioned to a web environment for refinement. Once the plan is finalized, it can be executed either in the remote cloud environment or within the local terminal.
